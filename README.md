# EventPulse

AI Powered Event Driven Traffic Forecasting System

# Problem Statement

Large scale events such as VIP movement, protests,
public gatherings and construction activities cause
localized traffic congestion.

Current systems rely heavily on manual planning.

EventPulse predicts traffic impact and recommends:

• Police deployment
• Barricade allocation
• Alternate routes
• Post mitigation analysis


# Features

 Risk Score Prediction
 Traffic Impact Estimation
 Police Requirement Recommendation
 Barricade Recommendation
 Diversion Planning
 Event Hotspot Visualization
 Learning Insights Dashboard
 Mitigation Effectiveness Analysis


# Dataset
